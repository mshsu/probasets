Mosaic Plot¶

Created mosaic plot type option

  • Only for two discrete variables.
  • Left and bottom axis show support, top and right axis show cumulative density
  • Bar width is marginal x-density, bar height is y|x density, color scale is marginal density of y
  • Bar area is joint density of x, y pairs
  • Color map is viridis - each color is a unique value in the support of y
In [5]:
xy = RV(Binomial(7, 0.5) * Binomial(10, 0.5)).sim(1000)
xy.plot(["mosaic"])

Distribution PDF Plot Shading¶

  • Changed .plot() to a chainable method for distributions

  • Added .shade() chainable method that shades areas under curve based on parameters (lt, gt, le, ge)

  • Shades area under curve if continuous distribution

  • Adds impulse lines under curve if discrete distribution

  • Adds impulse line at le/ge if specified and distribution is continuous

  • Returns error if lt/le value is less than gt/ge value

  • Returns error if both lt and le specified
  • Returns error if both gt and ge specified

Continuous Example:

In [4]:
